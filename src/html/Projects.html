Needle - Fast large file editor

Needle ~ ^w^ ~

Needle is a small, lightweight large file manipulation software for fast memory based modifications. meant to use to be used on those documents where large sudden changes are needed

How is works

Needle works by reading the document line by line into a strand in the systems memory,
Once that's completed it will completely purge the original document.
After all the starting actions are completed it will start reading each line and seeking the specified pattern with each strand
If a match is found it will replace the data in that section then write it back to document and remove the strand from the memory
If there isn't a match in the strand it will simple write it back and remove the strand from memory

Warnings

Needle will purge the original document!
Needle will write the strand in memory, meaning if the strand is 5 GB's and there's only 2 GB's of system memory your system will crash!
Needle will clear the terminal and use some CPU power!
